## Demo Application

You can build and install the module locally first in the project root folder.

```bash
$ mvn clean install
```

And you can build and run the [demo](demo) project:

```bash
$ cd demo
$ mvn clean verify && mvn -Pcargo.run
```

Visit http://localhost:8080/site/campaign1.
